## 数据类型

---

- 整数 `1, 100, -100, 0`

- 浮点型 `1.23, -0.9, 1.23e9, 1.2e-5`

- 字符串

  字符串可以使用单引号 `'abc'` 或双引号 `"abc"`，字符串内可使用斜杠 `\` 转义，可以使用 r 来禁用转义 `r'\\\'`。

  多行字符串可使用 `'''`，例如 

  ```
  print('''hello
  word
  ''')
  ```

- 布尔值 `True` , `False`

  布尔值相关运算

  ```
  True and True
  True or False
  not True
  ```

- 空值 `None`
